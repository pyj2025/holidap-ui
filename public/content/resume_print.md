# 왜 레주메 출력이 필요한가요?

캐나다에서는 walk-in 레주메 드랍(직접 방문하여 이력서 제출)이 일반적이므로, 프린트된 레주메가 필요합니다.

# 레주메 출력 방법 비교

## 1. 공공 도서관
- **장점**: 저렴한 비용, 안정적인 서비스
- **필요한 것**: 도서관 카드
- **비용**: 흑백 1장당 $0.15 내외

## 2. Work BC 센터 (BC주 한정)
- **장점**: 무료 프린트 및 구직 지원 서비스
- **필요한 것**: 구직자 등록 또는 예약
- **비용**: 무료

## 3. Staples 등 프린트샵
- **장점**: 편리함, 빠른 서비스
- **단점**: 상대적으로 비싼 비용

# 도서관 카드 발급 방법 (모든 지역 공통)

## 준비물
- 신분증 (여권, 비자 등)
- 거주 증명 (임대계약서, 우편물 등)

## 발급 절차
1. 가까운 도서관 지점을 방문하세요.
2. 프론트 데스크에서 회원 가입 신청서를 작성하세요.
3. 도서관 카드를 즉시 발급받을 수 있습니다.

# 지역별 상세 안내

## 토론토 - Toronto Public Library

[Toronto Public Library 공식 웹사이트](https://www.torontopubliclibrary.ca/)

### 프린트 비용
| 종류 | 1장당 비용 | 비고 |
| --- | --- | --- |
| 8.5" x 11" 흑백 | $0.15 | 양면은 두 페이지로 계산 |
| 8.5" x 14" 흑백 | $0.15 |  |
| 11" x 17" 흑백 (Toronto Reference Library 한정) | $0.20 |  |
| 컬러(모든 크기) | $0.55 |  |

### 도서관 카드가 없는 경우
- $1에 Quick Print 카드를 구매할 수 있습니다.
- 충전 금액은 환불, 이체가 불가능합니다.

### 결제 방법
- 온라인 ([토론토 도서관 프린트 서비스](http://print.tpl.ca))에서 credit/debit card로 결제 가능합니다.
- 도서관 방문 시 직원에게 현금, 신용카드, 직불카드로 충전 요청이 가능합니다.

### 프린트 방법
1. **도서관 컴퓨터에서 직접 프린트**
2. **원격 프린트**: [토론토 도서관 프린트 서비스](http://print.tpl.ca)에서 프린트 작업을 제출한 후 24시간 내 도서관을 방문해 출력하세요.

### 추가 서비스
- 스캔 서비스는 무료이며, 이메일로 전송 가능합니다.

### 주의사항
- 암호화된 PDF, Excel 파일 등은 원격 프린트 시 문제가 있을 수 있으니 도서관 컴퓨터 사용을 권장합니다.
- 프린트 계정에 충전된 금액이 있어야 하며, 프린터에서 도서관 카드 또는 Quick Print 카드 번호와 비밀번호를 입력 후 출력하세요.

## 밴쿠버 - Vancouver Public Library

[Vancouver Public Library 공식 웹사이트](https://www.vpl.ca/)

도서관 카드가 있으면 모든 지점에서 프린트 및 복사가 가능합니다.

### 프린트 비용
| 종류                         | 1장당 비용   |
|------------------------------|--------------|
| 8.5" x 11" 흑백 (단면)       | $0.20        |
| 8.5" x 11" 흑백 (양면)       | $0.30        |
| 8.5" x 11" 컬러 (단면)       | $0.75        |
| 8.5" x 11" 컬러 (양면)       | $1.00        |
| 11" x 17" 흑백 (단면)        | $0.40        |
| 11" x 17" 흑백 (양면)        | $0.60        |
| 11" x 17" 컬러 (단면)        | $1.50        |
| 11" x 17" 컬러 (양면)        | $2.00        |

### 결제 방법
- **동전 또는 복사카드**로 결제가 가능합니다.
  - 복사카드는 현장(서비스 데스크)에서 $2, $5, $10, $20 단위로 구매하세요.
- **결제 방법 결합 불가** (동전과 복사카드 동시 사용 불가)
- **동전 교환**은 서비스 데스크 또는 층별 교환기에서 가능합니다.

### 프린트 방법
1. **도서관 컴퓨터에서 직접 프린트**
   - 도서관 카드 또는 인터넷 접속 카드(Internet Access card)가 필요합니다.
   - 카드 번호와 PIN을 입력한 후 로그인하세요.
   - 파일을 열고 프린트 설정을 하세요 (기본: 흑백, 양면).
   - 컬러 프린트는 "Public-Colour on vpl-pcut"을 선택하세요.
2. **프린터에서 결제 및 출력**
   - 동전 또는 복사카드를 삽입하세요 (복사카드는 한 장만).
   - 프린터에서 카드 번호와 PIN을 입력하세요.
   - 인쇄 후 문서 및 잔돈/카드를 수령하세요.

### 추가 서비스
- 스캔 서비스는 무료이며, 이메일로 전송 가능합니다.

### 주의사항
- 프린트 작업 제출 후 5시간 내 미수령 시 삭제됩니다.
- 개인 기기(노트북, 스마트폰 등)에서 직접 프린트가 불가능합니다.
- 암호화된 PDF, Excel 파일 등은 도서관 컴퓨터에서만 출력하는 것을 권장합니다.
- **프린트 작업 당 결제 제한**: 동전 $10, 복사카드 $20 (초과 시 작업 분할 필요)
- 복사카드 유효기간은 없습니다.
