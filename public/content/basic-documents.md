# 기본준비물 (Document)

캐나다 입국하면서 필요한 기본 준비물입니다.

L자 파일에 모아서 기내용 짐으로 가지고 오시는것을 추천합니다.

- 여권
- 비자 최종 허가
- 잔고 증명서 : 은행 앱에서 발급 가능
- 보험 가입 확인서 : 보험 가입시 이메일로 전송
- (Optional) 비자 최종 허가서의 신체 검사 날짜와 1년 이상 입국일이 차이나는 경우 재검 확인서
- (Optional) 입국 후 바로 운전을 희망하시는 경우 국제운전면허증
