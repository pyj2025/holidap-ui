# 워홀러를 위한 레주메 작성 가이드

## 1. 레주메란?

레주메(Resume)는 캐나다 등 영어권 국가에서 구직 시 사용하는 이력서로, 본인의 경력, 학력, 기술, 자격 등을 간결하게 요약한 한 장짜리 문서입니다. 불렛포인트 중심의 간단명료한 작성이 특징입니다.

## 2. ChatGPT로 빠르게 레주메 만들기

ChatGPT를 활용하면, 내 경력과 지원할 직무를 입력하여 간단하게 레주메 초안을 만들 수 있습니다.

### 프롬프트 예시

![레주메 프롬프트.jpg](resume_guide_1.jpg)

```
나는 현재 캐나다에서 워킹홀리데이 비자로 체류 중이고, 바리스타나 리테일, 캐셔 등 고객 응대 중심의 서비스 직종에 지원할 예정이야.
아래 내 정보를 바탕으로 캐나다 현지 스타일의 한 페이지 분량 영어 이력서를 작성해줘.

비자 상태: 워킹홀리데이 비자 (오픈 워크퍼밋, 2026년 2월까지 유효)

경력:
- 바리스타 @Starbucks, 서울 (2022년 6월 ~ 2023년 1월)
- 판매보조(파트타임) @유니클로, 서울 (2021년 3월 ~ 2021년 12월)

학력: 경영학 전공, 한국 소재 대학교 재학 중 (현재 휴학 중)
언어: 한국어(모국어), 영어(기본 회화 가능)
소프트 스킬: 친절함, 빠른 적응력, 멀티태스킹, 고객 응대에 강함
근무 가능일: 즉시 시작 가능, 평일/주말 모두 가능
```

### 프롬프트 템플릿

```
아래 내 경력과 지원 직무를 참고해서 캐나다 워킹홀리데이용 레주메 불렛포인트를 영어로 작성해줘.

비자 상태: 워킹홀리데이 비자 (오픈 워크퍼밋, [유효기간]까지 유효)

경력:
- [직무] @ [회사명], [도시/국가] ([시작 년/월 ~ 종료 년/월])
- [직무] @ [회사명], [도시/국가] ([시작 년/월 ~ 종료 년/월])

학력: [전공], [학교명], [국가] (예: 현재 휴학 중 / 졸업 등)
언어: [예: 한국어(모국어), 영어(기본 회화 가능)]
소프트 스킬: [예: 친절함, 빠른 적응력, 멀티태스킹, 고객 응대에 강함]
근무 가능일: [예: 즉시 시작 가능, 평일/주말 모두 가능]
```

### 사용 방법

- ChatGPT가 불렛포인트, 요약, 키워드 중심의 레주메 문장을 자동으로 생성해줍니다.
- 생성된 내용을 복사해 템플릿에 붙여넣고, 본인 상황에 맞게 수정하세요.

## 3. 추천 Resume 템플릿 사이트

- [Microsoft Create Resume Templates](https://create.microsoft.com/en-us/templates/resumes): 다양한 직종별 무료 레주메 템플릿 제공, Word에서 바로 편집 가능
- [Canva Resume Templates](https://www.canva.com/resumes/templates/): 디자인이 깔끔한 무료/유료 템플릿 다수, 온라인 편집 지원

## 5. 레주메 작성 팁

- 불렛포인트로 간결하게 작성하세요.
- 지원하는 직무에 맞게 레주메를 여러 버전으로 준비하세요.
- 현지 연락처와 비자 상태 등 필수 정보를 명확히 표기하세요.
- 최종 제출 전 오타, 연락처, 이메일 등을 꼼꼼히 다시 확인하세요.
- 업무 내용은 맡았던 역할과 책임 중심으로 간결하게 작성하세요.
- 가능하면 하루 고객 수, 매출액, 팀원 수 등의 숫자를 넣으면 더 신뢰감이 생깁니다.
- 숫자를 넣기 어려울 때는 경험의 폭과 강점을 명확히 표현하는 데 집중하세요.
- 서비스직은 보통 워크인 지원(직접 방문 지원)이 많으므로, 너무 복잡한 디자인보다는 깔끔하고 기본에 충실한 레주메를 추천합니다.

## 6. Canva 템플릿 사용에 대하여

Canva 이력서 템플릿은 ATS(자동 서류 분류 시스템)에 최적화되어 있지는 않지만, 서비스업(바리스타, 리테일 등)에서는 워크인 지원하는 경우가 많기 때문에 충분히 유용합니다. 단, 사무직이나 대기업 온라인 지원 시에는 워드 혹은 PDF 기반의 텍스트 중심 레이아웃을 추천합니다.
