# 카드 사용

![image.png](/images/atm-travel-card-info/image.png)

대부분 한국에서 발급되는 여행용 카드는 contactless 심볼이 있고, 캐나다에서 tap to pay로 카페, 레스토랑, 마트 등에서 문제 없이 사용할 수 있습니다.

인터넷 쇼핑몰의 경우 일부 한국 카드 결제가 막혀있으면 사용이 안될 수 있습니다. 애플페이의 경우에도 한국 카드는 등록이 되지 않습니다.

# ATM 출금

캐나다에서 수입이 있기 전까지의 rent를 내기 위해 일반적으로 ATM에서 달러를 뽑은 후, 현지 은행에 입금하여 e-transfer를 하게 됩니다. 대부분의 카드에서 ATM 출금 수수료가 없다고 광고하지만, 카드사의 수수료나 국제 브랜드는 물론 현지 ATM의 수수료도 있기 때문에 아무 ATM에서 뽑으면 surcharge가 발생할 수 있습니다. 캐나다의 많은 신용 조합들 (대표적으로 Meridian Credit Union나 VanCity)이 이 수수료 없이 현금 인출이 가능한 Exchange Network에 가입되어 있어서 이러한 은행들의 ATM이나 한국계 은행들의 ATM에서 출금하는 것을 추천드립니다.

| ATM | 수수료 없는 카드 |
| --- | --- |
| Vancity | 트래블월렛, 토스뱅크, SOL트래블, 트래블로그, 트래블GO |
| 캐나다 하나은행 | SOL트래블, 트래블로그 (마스터) |
| 캐나다 신한은행 | 토스뱅크, 트래블러스 |

은행별로 수수료 무료에 금액/횟수 제한이 걸려있는 경우가 있고, 정책이 업데이트 될 수 있으니 확인이 필요합니다.

현지 ATM별로 한번에 뽑을 수 있는 금액에 제한이 걸려 있는 경우가 많이 있습니다. 한국 트래블 카드의 ATM 인출 무료 횟수 제한과 교차해서 본인이 원하는 금액을 매달 뽑을 수 있는지 확인하세요.

카드가 고장나거나 분실했을 경우, 대부분의 카드사가 해외 배송을 해주지 않습니다. 만약을 대비해 두개 이상의 카드를 가져오시는 것을 추천합니다. 

평일 업무시간 외에 인출할 경우 수수료가 부과되는 케이스가 있으니 가능하면 평일 업무시간에 인출하시길 추천드립니다.

카드들이 지원하는 한도 이상의 출금이 필요하실 경우, 송금 서비스를 별도로 이용하시는 것을 추천드립니다. 송금 서비스에 대한 안내는 다음을 참고하세요:

# 트래블카드 종류

Disclaimer: 모든 정보는 작성 시점 기준이며 사용 전 확인이 필요합니다.

last updated at 2025-06-24

## 트래블월렛

- 트래블월렛 앱에서 환전 후 이용
- 해외 배송 불가
- 환전수수료 0.25~0.5%
- ATM 인출 금액 월 USD 500 초과시 초과금액의 2% 수수료
- ATM 출금 1회 한도 USD 400, 1일 한도 USD 1,000, 월간 한도 USD 2,000
- 월 USD 500까지 카드사 출금 수수료 면제 (이후 2%)
- 카드결제 연간한도 USD 100,000

## 토스뱅크

- 외환 통장에 환전 후 이용 or 자동 환전 기능 사용
- 해외배송불가
- CAD 보유 한도 없음
- 캐나다 달러 환전 수수료 0%
    - 원화로 재환전시 수수료 0%
- ATM 출금 월 5회 무료 (매월 1일 초기화). 단, 월 USD 700 초과 시 횟수 상관없이 수수료 발생.
    - 출금 횟수 한도(5회)를 금액 한도(700 USD) 보다 먼저 초과하거나 동시에 초과하는 경우
        - 누적 인출 금액 관계 없이, 출금 6회차 거래부터 인출 금액의 1% + USD 3 부과
    - 금액 한도(700 USD)를 출금 횟수 한도(5회) 보다 먼저 초과하는 경우
        - 누적 인출 금액 700 USD을 초과하는 시점의 거래: 700 USD초과 차액의 1% + 3 USD부과
        - 상기 이후 거래: 인출 금액의 1% + 3 USD 부과

## 트래블로그, 트래블GO (하나은행)

- 하나머니 앱에서 환전한 외화를 카드에 충전하여 사용 (하나머니 회원가입 필수)
- 고장/분실시 해외배송 가능
- 마스터카드, 비자카드 두 종류
- 캐나다 달러 환전 수수료: 0
- 보유한도: 약 200만원 (특별한도 신청 시 300만원으로 상향 가능)
- 원화 재환전 시 수수료: 1%
- 해외 ATM 수수료 면제: 해외인출 수수료 - 인출건당 $3 + 국제브랜드 수수료 - 이용금액의 1%
- 결제한도: USD 5,000/일, USD 10,000/월
- ATM 인출한도: USD 6,000/일, USD 10,000/월

## SOL트래블 (신한은행)

- 외화예금 통장에 미리 환전해두고, 체크카드로 해외에서 바로 결제하거나 ATM에서 인출하여 사용
- 해외 배송 불가
- 캐나다 달러 환전 수수료: 0
- 원화 재환전 시 수수료: 0
- 보유한도: 약 6,500만원, 결제한도: 약 780만원
- ATM 출금한도 USD 10,000/월
- 해외 수수료 관련 유의사항
    - 해외 가맹점 결제 수수료 면제: 이용금액의 1.2%
    - 해외 ATM 수수료 면제: 해외인출 수수료 - 인출건당 $3 + 국제브랜드 수수료 - 이용금액의 1%
    - 현지 ATM운영사가 부과하는 수수료는 지원하지 않음

## 트래블러스 (국민은행)

- KB pay 앱에서 외화 머니로 환전 후 이용
- 분실 / 고장시 해외 배송 불가
- 보유 한도: 원화 기준 200만원
- 연간 충전 가능 한도 USD 100,000
- 환전수수료 연간 USD 100,000원까지 0%
    - 원화로 재환전씨 환급수수료 1%
- 월 10회 무료. 금액 제한 X
    - 10회 초과시 국제브랜드수수료 1% + 해외서비스수수료 건당 3$

## 트래블카드 사용 시 유의점

트래블카드는 환율이 유리할 때 미리 환전해두고, 환전 우대 혜택을 받아 캐나다에서 바로 결제하거나 ATM 출금할 수 있다는 장점이 있습니다. 하지만 캐나다 달러 외화예금에는 이자가 붙지 않는 경우가 많기 때문에, 장기간 또는 큰 금액을 보관할 계획이라면 주의가 필요합니다. 환전 시기와 금액을 신중하게 계획하는 것이 좋습니다.
