# 캐나다 유심 신청 및 현지 사용

# [수퍼셀](https://www.supercellhub.com/)

**수퍼셀 (카카오톡 아이디: SCELL)**

- **가입 방법**:
  카카오톡 채널을 를 통해 문의
  → 캐나다 통신사별 **요금제**, **프로모션** 등 정보 제공
  → 옵션 비교 후 선택, **신청서 제출**하면 가입 진행

# [유심마을](https://smartstore.naver.com/canadausim/products/6491443570)

- 네이버 스마트스토어에서 요금제, 프로모션 비교 및 유심 구매
- 구매 페이지에 있는 링크에서 가입 폼 작성
- [카톡 채널](pf.kakao.com/_iBPxms/chat)에 이름 남기기

# **유의사항**:

- 통신사별 프로모션은 **가입 시점**에 따라 달라질 수 있음
- eSIM / 물리 SIM 카드 모두 사용 가능
- eSIM의 경우 적용 기종 확인
